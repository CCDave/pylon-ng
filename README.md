#PYLONG-NG

pylon-ng 并不是一个典型的MVC框架!

好的框架，应该是：

* 概念越少越好
* 功能特性可以扩展
* 基于本质的抽象，稳定，易理解
* 

## 快速开始
```
  mkdir ~/devspace ;
  git clone  git@github.com/xcodecraft/pylon-ng
  rg tpl -t ~/devspace/pylon-ng/src/template -o ~/devspace/<you-prj>
```
